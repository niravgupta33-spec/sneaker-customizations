# 👟 KickCraft — Premium Sneaker Customizer

A premium 3D sneaker customization website built with HTML, CSS, JavaScript, and Three.js.

## ✨ Features

- **3D Sneaker Viewer** — Interactive Three.js-powered 3D model viewer with orbit controls
- **Real-time Customization** — Pick colors, materials, and finishes in real-time
- **Material Studio** — Explore premium materials (leather, suede, mesh, etc.)
- **Shop & Cart** — Browse featured sneakers with a fully functional cart system
- **Checkout Simulation** — Simulated payment gateway with order history
- **Responsive Design** — Beautiful dark-themed UI optimized for all screen sizes

## 🚀 Deployment

This project is configured for **Vercel** static deployment:

1. Push to GitHub
2. Import the repository in [Vercel](https://vercel.com)
3. Deploy — no build step needed!

## 🛠 Tech Stack

- **HTML5** + **CSS3** (custom properties, glassmorphism, gradients)
- **Vanilla JavaScript**
- **Three.js** (3D rendering, OBJ/GLB model loading)
- **Google Fonts** (Outfit)
